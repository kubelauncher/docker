# kubelauncher/docker

Community-maintained Docker images built on Alpine Linux. Multi-arch (`amd64`/`arm64`), non-root (UID 1001).

## Images

| Image | Description |
|-------|-------------|
| [redis](images/redis) | Redis in-memory data store |
| [postgresql](images/postgresql) | PostgreSQL relational database |
| [rabbitmq](images/rabbitmq) | RabbitMQ message broker |
| [kubectl](images/kubectl) | kubectl CLI utility |
